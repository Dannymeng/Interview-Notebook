# 04 - 二维数组中的查找 

## 题目描述
给定一个二维数组，其每一行从左到右递增排序，从上到下也是递增排序。给定一个数，判断这个数是否在该二维数组中。

```
004Consider the following matrix:
[
  [1,   4,  7, 11, 15],
  [2,   5,  8, 12, 19],
  [3,   6,  9, 16, 22],
  [10, 13, 14, 17, 24],
  [18, 21, 23, 26, 30]
]

Given target = 5, return true.
Given target = 20, return false.
```

## 解决方案

思路：

1、暴力法，时间复杂度高：O（n^2）

2、利用递增序列的特点，从二维数组的右上角开始遍历

- 当前数值比所求的数要小时，向下
- 当前数值比所求的数要大时，向左

时间复杂度：O（m+n）

```java
public class Solution {
    public boolean Find(int target, int [][] array) {
        int a1 = array.length; // a1列
        int b1 = 0;
        while (a1 - 1 >= 0 && b1 < array[0].length) {
            if (array[a1 - 1][b1] > target) {
                a1 -= 1;
            } else if (array[a1 - 1][b1] < target) {
                b1 += 1;
            }else {
                return true;
            }
        }
        return false;
    }
}
```
